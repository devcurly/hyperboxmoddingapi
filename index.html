Getting Started

Learn how to create mods for HyperBox using the Lua scripting API.

Installation

  1. Navigate to your game directory: GAME/ALS_EU/Mods/
  2. Create a new .lua file directly in the Mods folder (not in a subfolder)
  3. Additional assets (sounds, models) can be placed in subfolders
  4. Start the game to load your mod
Important
Lua script files must be placed directly in the Mods folder. Nested folders are not supported for script files, but can be used for assets like audio and 3D models.

Basic Script Structure

Example: hello_world.lua
-- Initialize your mod
function mod:begin_play()
    Print("Hello World!", 3)
    Message("Mod loaded successfully", 1, 3)
end

-- Handle key press events
function mod:key_pressed(key)
    if key == "G" then
        Print("G key pressed", 2)
    end
end

-- Update loop (runs every frame)
function mod:tick(delta)
    -- Your update logic here
    -- delta = time since last frame
end

Note on While Loops
Standard while loops don't work the same as in vanilla Lua. Use mod:tick(delta) for continuous execution or Delay() for timed events.

Vector Operations

Vectors represent 3D positions and rotations with x, y, z components. Access individual components with .x, .y, .z.

Constructor

NewVector(x, y, z)
Creates a new vector with optional x, y, z components. Defaults to (0, 0, 0) if no parameters provided.
Parameters
x float X component (optional, default: 0)
y float Y component (optional, default: 0)
z float Z component (optional, default: 0)
Example
local origin = NewVector()        -- (0, 0, 0)
local position = NewVector(5, 5, 5) -- (5, 5, 5)
local x = position.x               -- Access x component

Arithmetic Operations

AddVector(a, b)
Adds two vectors component-wise.
Example
local result = AddVector(NewVector(1,1,1), NewVector(2,2,2))
-- result = (3, 3, 3)
SubtractVector(a, b)
Subtracts vector b from vector a component-wise.
Example
local result = SubtractVector(NewVector(2,2,2), NewVector(1,1,1))
-- result = (1, 1, 1)
MultiplyVector(a, b)
Multiplies two vectors component-wise.
Example
local result = MultiplyVector(NewVector(10,10,10), NewVector(2,1,3))
-- result = (20, 10, 30)
MultiplyVectorF(vector, scalar)
Multiplies a vector by a scalar float value.
Example
local result = MultiplyVectorF(NewVector(1,1,1), 5)
-- result = (5, 5, 5)
DivideVector(a, b)
Divides vector a by vector b component-wise.
Example
local result = DivideVector(NewVector(10,10,10), NewVector(2,1,3))
-- result = (5, 10, 3.333...)
DivideVectorF(vector, scalar)
Divides a vector by a scalar float value.
Example
local result = DivideVectorF(NewVector(10,10,10), 2)
-- result = (5, 5, 5)

Direction Vectors

ForwardVector(object)
Returns the forward direction vector of the given object.
Parameters
object Object The object to get the forward vector from
UpVector(object)
Returns the up direction vector of the given object.
RightVector(object)
Returns the right direction vector of the given object.

Transform Functions

GetLocation(object)
Returns the world location of the given object as a vector.
SetLocation(object, vector)
Sets the world location of the given object.
Parameters
object Object The object to move
vector Vector The new world location
GetRotation(object)
Returns the world rotation of the given object as a vector.
SetRotation(object, vector)
Sets the world rotation of the given object.

Math Functions

Mathematical utilities for calculations, interpolation, and distance measurements.

Utility Functions

Clamp(input, min, max)
Clamps a float value between minimum and maximum bounds.
Example
local result = Clamp(75, 0, 50)
-- result = 50
MapRange(input, inRangeA, inRangeB, outRangeA, outRangeB)
Maps a value from one range to another range.
Example
local result = MapRange(50, 0, 100, 0, 500000)
-- result = 250000 (50% of output range)
Distance(vectorA, vectorB)
Returns the Euclidean distance between two vectors.
Magnitude(vector)
Returns the magnitude (length) of a vector.
Example
local mag = Magnitude(NewVector(1, 1, 1))
-- mag â‰ˆ 1.732

Interpolation

LerpV(current, target, speed)
Linear interpolation between two vectors. Smoothly transitions from current to target.
Parameters
current Vector Starting vector
target Vector Target vector
speed float Interpolation speed (0-1)
LerpF(current, target, speed)
Linear interpolation between two float values.
NearlyEqualV(vector, equalTo, tolerance)
Checks if two vectors are approximately equal within a tolerance.
NearlyEqualF(float, equalTo, tolerance)
Checks if two floats are approximately equal within a tolerance.

Rotation Utilities

LookAtRotation(from, to)
Calculates the rotation needed to look from one position towards another.
ForwardVectorFromRot(rotation)
Extracts the forward direction vector from a rotation vector.

Player API

Functions for accessing player and camera information.

Player Functions

GetPlayer()
Returns the player character object.
Example
local player = GetPlayer()
local playerPos = GetLocation(player)
CameraLocation()
Returns the current camera world location as a vector.
CameraForwardVector()
Returns the forward direction vector of the camera. Useful for spawning objects in front of the player.
Example
local camPos = CameraLocation()
local camForward = CameraForwardVector()
local spawnPos = AddVector(camPos, MultiplyVectorF(camForward, 300))

Character API

Control and manipulate character properties, physics, and behavior.

Character Management

CreateCharacter(location, rotation, scale, type)
Spawns a new NPC character at the specified location.
Parameters
location Vector World spawn position
rotation Vector Initial rotation
scale Vector Character scale
type string NPC type identifier
GetCharacters()
Returns a table of all currently spawned characters in the world.
GetMesh(character)
Returns the skeletal mesh component of the character (their body/ragdoll).
Kill(character, instant)
Kills the specified character.
Parameters
character Character The character to kill
instant bool If true, kills instantly without animation
Damage(character)
Applies damage to the specified character.

Character Properties

GetProperties(character)
Returns a table containing all properties of the character.
SetProperty(character, property, value)
Sets a specific property of the character.
Example
SetProperty(GetPlayer(), "Health", 100)
SetProperty(npc, "MaxHealth", 200)
SetSkinColor(character, red, green, blue)
Sets the character's skin color using RGB values (0-1 range).
ToggleHair(character, visible)
Toggles hair visibility for the character.
GetOverlayState(character)
Returns the character's current overlay animation state.

Movement & Stance

SetStance(character, stance)
Changes the character's stance.
Valid Stances
"Standing" Normal standing position
"Crouching" Crouched position
SetGait(character, gait)
Changes the character's movement speed.
Valid Gaits
"Walking" Slow walk speed
"Running" Normal running speed
"Sprinting" Maximum speed

Physics & Ragdoll

ToggleRagdoll(character)
Toggles ragdoll physics on/off for the character.
TogglePunching(character)
Toggles the fighting/punching mechanic on/off.
AddForce(mesh, bone, force)
Applies a continuous force to a specific bone on the character mesh.
AddForceAtLocation(mesh, bone, force, location)
Applies a force at a specific relative location on a bone.
AddImpulse(mesh, bone, impulse)
Applies an instant impulse to a bone (one-time force).
AddImpulseAtLocation(mesh, bone, impulse, location)
Applies an instant impulse at a specific relative location.
GetMeshPartVelocity(mesh, bone)
Returns the current velocity of a specific body part.
SetMeshPartVelocity(mesh, bone, velocity)
Directly sets the velocity of a specific body part.

Sockets

GetSocketLocation(mesh, socketName)
Returns the world location of a socket on the character mesh.
GetSocketRotation(mesh, socketName)
Returns the world rotation of a socket on the character mesh.

Stumble Settings

GetStumbleSettings(character)
Returns a table of the character's stumble physics settings.
SetStumbleSettings(character, setting, value)
Modifies a specific stumble setting value.
StumbleOverride(character, setting, enabled)
Enables or disables override between custom and vanilla stumble settings. Required when using custom stumble settings.

World API

Functions for creating objects, playing sounds, and manipulating the game world.

Object Creation

CreateShape(shape, location, rotation, size, material)
Creates a primitive shape in the world.
Parameters
shape string Shape type (e.g., "Cube", "Sphere", "Cylinder")
location Vector World spawn position
rotation Vector Initial rotation
size Vector Object scale
material Material Material created with NewMaterial()
NewMaterial(r, g, b, metallic, specular, glow, roughness)
Creates a new material with specified properties.
Parameters
r, g, b float (0-1) RGB color values
metallic float (0-1) Metallic property
specular float (0-1) Specular reflectivity
glow float (0-10000) Emissive glow strength
roughness float (0-1) Surface roughness
CreateMesh(filePath, location, rotation, scale, enableCollision)
Imports and spawns a custom mesh from an FBX file at runtime.
File path is relative to the Mods folder. Example: "MyAssets/model.fbx"
CreateText(location, rotation, scale, material, text)
Creates 2D text in the 3D world.
SetTextColor(textObject, r, g, b)
Sets the color of a text object using RGB values (0-1 range).

Audio

PlaySound(filePath, location, minDist, maxDist, volume, pitch, autoDestroy, is3D)
Imports and plays a sound from a WAV file.
Parameters
filePath string Path relative to Mods folder
location Vector 3D sound origin
minDist float Minimum attenuation distance
maxDist float Maximum audible distance
volume float (0-1) Volume level
pitch float (0-2) Pitch modifier
autoDestroy bool Destroy after playing
is3D bool Enable spatial audio

Object Manipulation

SetSize(object, vector)
Sets the scale of an object.
GetSize(object)
Returns the current scale of an object as a vector.
SetVisibility(object, visible)
Shows or hides an object/actor.
ToggleCollision(object, enabled)
Enables or disables collision on an object. Optional boolean parameter.
TogglePhysics(object, enabled)
Enables or disables physics simulation on an object. Optional boolean parameter.
GetVelocity(object)
Returns the current velocity of an object as a vector.
SetVelocity(object, velocity)
Directly sets the velocity of an object.
GetMass(object)
Returns the object's mass in kilograms.
SetMass(object, mass)
Sets the object's mass in kilograms.

Actor Relationships

AttachActors(object1, object2, bone)
Attaches one object to another. The bone parameter is only required when attaching to a mesh.
DetachActor(object)
Detaches an actor from its parent.
Group(actorsTable)
Combines multiple actors into a single grouped actor.
UnGroup(object)
Separates a grouped actor and returns a table of all individual actors.

Ray & Overlap Tracing

LineTrace(show, ignorePlayer, start, end, actorsToIgnore)
Performs a line trace from start to end position, returning hit information.
Parameters
show bool Visualize the trace line
ignorePlayer bool Ignore player in trace
start Vector Starting position
end Vector Ending position
actorsToIgnore table Array of actors to ignore
SphereTrace(show, ignorePlayer, start, end, radius, actorsToIgnore)
Performs a sphere trace with specified radius from start to end position.
CreateOverlapEvent(location, rotation, scale, eventName)
Creates a box trigger volume that fires an event when entered or exited.
RegisterTouch(object, eventName)
Binds an object's touch event to a custom event handler.

World State

GetTimeDilation()
Returns the current time dilation (game speed) as a float between 0 and 2.
SetTimeDilation(dilation)
Sets the time dilation (game speed). Values range from 0 (paused) to 2 (double speed).
GetWorldTimeDelta()
Returns the delta time (time since last frame) in seconds.
GetWorldTime()
Returns the current world time as a float between 0 and 2400 (24-hour format).
SetWorldTime(time)
Sets the world time to a value between 0 and 2400.
RandomPoint(location, radius)
Returns a random reachable point within a specified radius. Useful for AI navigation and random spawning.

UI & Messaging

Print(message, duration)
Displays a message in the top-left corner of the screen.
Parameters
message string Text to display
duration float Display time in seconds
Message(message, typeSpeed, duration)
Displays a typewriter-style message at the top of the screen.
Parameters
message string Text to display
typeSpeed float Characters per second
duration float Display time in seconds
Delay(time, callback)
Executes code after a specified delay.
Example
Delay(3, function()
    Print("3 seconds have passed", 2)
end)

AI API

Control NPC behavior, movement, and combat actions.

Movement & Navigation

AIMoveTo(character, location)
Commands an AI character to move to the specified location using pathfinding.
AIGetMoveStatus(character)
Returns the current status of the AI's movement task.
AIStopMovement(character)
Immediately stops the character's movement.

Combat

AIFire(character)
Makes the character fire their equipped weapon, or punch if unarmed.
AIToggleAim(character, target, bone)
Toggles aiming at a target character. Optional bone parameter to aim at a specific body part.
Parameters
character Character The AI character
target Character The target to aim at
bone string Specific bone to aim at (optional)

Equipment

AIEquip(character, random, gunName)
Equips a weapon to the AI character. Can randomize or specify a specific weapon.
Parameters
character Character The AI character
random bool Randomize weapon selection (optional)
gunName string Specific weapon name (optional)
AIUnEquip(character)
Unequips the character's current weapon.

Alert State

AIAlert()
Alerts all AI in the world. Makes civilians scared or angry, cops hostile, and alerts enemies to the player's presence.

Event Callbacks

Event handlers that respond to game events and user input.

Lifecycle Events

function mod:begin_play()
Called once when a level is loaded. Use this to initialize your mod.
Example
function mod:begin_play()
    Print("Mod initialized!", 3)
    -- Initialize variables
    -- Spawn initial objects
end
function mod:tick(delta)
Called every frame. Use for continuous updates and monitoring.
Parameters
delta float Time elapsed since last frame in seconds
Example
function mod:tick(delta)
    -- Runs every frame
    -- Use delta for frame-independent calculations
    local player = GetPlayer()
    local pos = GetLocation(player)
end
function mod:CustomLevelLoaded(levelName)
Called when a custom level is loaded. Returns the level name.
Parameters
levelName string Name of the custom level
Example
function mod:CustomLevelLoaded(levelName)
    if levelName == "MyCustomMap" then
        -- Level-specific initialization
    end
end

Input Events

function mod:key_pressed(key)
Called when a keyboard key is pressed.
Parameters
key string The key that was pressed
Example
function mod:key_pressed(key)
    if key == "F" then
        Print("F key pressed", 2)
    elseif key == "G" then
        local player = GetPlayer()
        Kill(player, true)
    end
end
function mod:key_released(key)
Called when a keyboard key is released.
Parameters
key string The key that was released

Game Events

function mod:npc_died(npc)
Called when an NPC dies.
Parameters
npc Character The NPC that died
Example
function mod:npc_died(npc)
    local pos = GetLocation(npc)
    Print("NPC died at position: " .. tostring(pos), 2)
end
function mod:player_died()
Called when the player dies.
Example
function mod:player_died()
    Print("Player has died!", 3)
    -- Handle player death
end
